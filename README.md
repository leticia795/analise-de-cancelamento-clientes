# 📊 Previsão de Cancelamento de Clientes

## 📌 Sobre o Projeto

Este projeto tem como objetivo prever se um cliente irá cancelar um serviço com base em variáveis como tempo de relacionamento e quantidade de clientes.

A análise foi realizada utilizando técnicas de Machine Learning, com foco em classificação.

---

## 🎯 Objetivo

Identificar padrões de cancelamento e prever o comportamento dos clientes para apoiar decisões estratégicas de retenção.

---

## 🚀 Tecnologias Utilizadas

* Python
* Pandas
* Scikit-learn
* Matplotlib

## 📊 Metodologia

1. Leitura e análise dos dados
2. Separação entre treino e teste
3. Treinamento com Regressão Logística
4. Avaliação do modelo com métricas

---

## 📈 Resultados

* Acurácia: 80%
* Boa precisão na identificação de cancelamentos
* Recall moderado (o modelo pode melhorar na detecção de todos os cancelamentos)
